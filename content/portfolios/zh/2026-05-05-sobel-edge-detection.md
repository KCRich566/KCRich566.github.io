### 項目概述
實作一個純前端邊緣偵測應用程式，用於比較常見的電腦視覺邊緣算子。使用者可以在瀏覽器載入圖片，程式會在本機完成灰階轉換、模糊前處理、邊緣計算與 Canvas 視覺化，不需要後端服務。

### 技術實現

#### 支援方法
- **Sobel**: 使用水平與垂直卷積核計算梯度
- **Canny**: 包含模糊、梯度、非極大值抑制、雙閾值與 hysteresis 的多階段流程
- **Prewitt**: 較輕量的梯度算子，適合一般邊緣萃取
- **Scharr**: 比 Sobel 具備更佳旋轉對稱性的梯度算子
- **Laplacian**: 使用二階導數回應快速亮度變化

#### 核心功能
1. **圖像處理管線**
   - 瀏覽器本機載入圖片
   - 灰階轉換
   - 可選擇模糊前處理
   - 依方法執行邊緣計算
   - 依閾值輸出邊緣結果

2. **進階處理選項**
   - 可調整高閾值
   - 可調整 Canny 低閾值
   - 可切換反相顯示
   - 可下載 Canvas 輸出結果

3. **視覺化與輸出**
   - 原圖與結果並排顯示
   - 可即時切換方法與參數
   - 下載 PNG 結果圖

### 技術堆疊
- **程式語言**: JavaScript, HTML, CSS
- **瀏覽器 API**: Canvas, File API, Blob/Object URL
- **架構**: Jekyll include 搭配獨立 CSS 與 JavaScript

### 性能指標
- **處理方式**: 完全在瀏覽器執行
- **隱私**: 圖片保留在本機，不會上傳
- **輸入大小**: 大圖會縮放到適合瀏覽器互動處理的尺寸

### 應用場景
- **機器視覺系統**: 製造業品質檢查
- **光學字符識別 (OCR)**: 文字邊界檢測
- **物體偵測**: 深度學習模型的前置處理
- **醫學影像**: 解剖結構邊界識別
- **自動駕駛**: 道路車道和障礙物檢測
- **文件分析**: 文字區域提取

### 主要成就
- 整合多種常用邊緣偵測方法
- 在瀏覽器中實作實用版 Canny 流程
- 提供閾值、模糊與顯示模式控制
- 保持純前端架構，適合 GitHub Pages 部署

### 成果
完成一個可重複使用的邊緣偵測工具，能在靜態作品頁中直接展示核心電腦視覺技術，並保留未來擴充更多濾波器或視覺化模式的彈性。
