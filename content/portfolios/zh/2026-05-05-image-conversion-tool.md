### 項目概述
開發專業級的圖片轉檔應用程式，支援靈活的處理模式 - 包括單一檔案轉檔和批次處理功能，可進行大規模圖像操作。

### 主要功能
- **單次轉檔**: 轉換單一圖片檔案到多種格式
- **批次處理**: 在一次操作中高效處理多張圖片
- **格式支援**: 
  - JPEG, PNG, BMP, TIFF, GIF
  - WebP, HEIC 等多種格式
- **圖片參數調整**:
  - 解析度/縮放
  - 品質調整
  - 色彩空間轉換
  - 壓縮最佳化
- **性能優化**: 批次操作採用多線程處理
- **使用者友善介面**: 直觀的 CLI 和 GUI 版本

### 技術堆疊
- **程式語言**: C#, Python
- **函式庫**: OpenCV, Pillow, .NET Image APIs
- **架構**: 
  - 單一檔案處理器
  - 批次處理器（含任務隊列）
  - 可配置的處理管線

### 應用場景
- 專業攝影師: 大量圖片格式轉檔以供歸檔
- 網頁開發者: 優化圖片以用於網站部署
- 系統管理員: 批次媒體檔案處理
- 內容創作者: 為多個平台進行格式轉換

### 性能表現
- 單一檔案: 典型圖片 < 1 秒
- 批次處理: 可處理 1000+ 張圖片並自動最佳化
- 記憶體效率: 大檔案採用串流處理

### 成果
開發完成的強大、可在生產環境中部署的圖片轉檔工具，已在多個專業環境中用於日常批次處理操作。
